HTML Commands for home integration

Actions:

Address Description
http://[ip_of_pill_dispenser]/alarm Play the default alarm no 1
http://[ip_of_pill_dispenser]/alarm?id=[custom_id] Play the alarm "id"
http://[ip_of_pill_dispenser]/dispense Dispense the current tray in alert, does nothing if there's no alarm
http://[ip_of_pill_dispenser]/droput?tray=[number_of_tray] Drop the pill in tray "number" (1 to 10)
http://[ip_of_pill_dispenser]/dismiss Dismiss current alarm that are in progress
http://[ip_of_pill_dispenser]/fix?tray=[number_of_tray] Make half step to fix tray "number" (1 to 10)

Information:

Address Description
http://[ip_of_pill_dispenser]/tr_name?id=[number_of_tray] Return name of the tray
http://[ip_of_pill_dispenser]/tr_color?id=[number_of_tray] Return 0 to 7 for assigned color (check color table)
http://[ip_of_pill_dispenser]/tr_inst?id=[number_of_tray] Return 1 if tray is installed
http://[ip_of_pill_dispenser]/tr_enab?id=[number_of_tray] Return 1 if tray is enabled for daily Alerts
http://[ip_of_pill_dispenser]/tr_trig?id=[number_of_tray] Return 1 if tray is triggered (Alert in progress)
http://[ip_of_pill_dispenser]/tr_disp?id=[number_of_tray] Return 1 if tray is dispensed for today
http://[ip_of_pill_dispenser]/tr_dism?id=[number_of_tray] Return 1 if tray is dismissed for today
http://[ip_of_pill_dispenser]/tr_time?id=[number_of_tray] Return alarm time for this tray

Color Table:

ID RGB Color Color Name

4 0, 255, 255 CYAN
5 0, 255, 0 GREEN
